Проект BestCS ("Наилучшее из C#") состоит в подборке из нескольких библиотек, находящихся в статусе "открытого проектного кода", объединённых под общим пространством имён для удобства компиляции и использования. В этот проект на данный момент включены следующие сборки:
Aforge.Net
OpenPop
OpenSmtp
Docx
NAudio
GalaMvvmLight
Newtonsoft.Json
